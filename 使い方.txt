QZSS 受信機 セットアップ手順(Windows)
========================================

【1】初回だけ: Pythonパッケージのインストール
このフォルダで以下を実行してください。

    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt

【2】受信機のCOMポートを確認
デバイスマネージャー →「ポート(COM と LPT)」で、受信機を挿した状態で
表示される名前(例: COM3)を確認してください。

【3】実行(一発起動、重要情報+注意情報の両方を送信)
COMポート名を指定して、以下を実行するだけです。

    start_receiver.bat COM3

(ボーレートは省略時9600です。必要なら第2引数で指定: start_receiver.bat COM3 115200)

これで、緊急地震速報・震源・震度速報・津波・Jアラートなどの「重要情報」は
メイン地図(https://qzss-map-85436528666.asia-northeast1.run.app/ )へ、
台風・気象警報・海上警報などの「注意情報」は
注意情報マップ(https://qzss-map-caution-85436528666.asia-northeast1.run.app/ )へ、
自動で振り分けて送信されます。

【4】動作確認したい時
read_legacy_dual.py 実行中のターミナルでEnterキーを押すと、
両方にテスト通報を送信できます。もう一度Enterを押すと
重要情報側の取消(終了)信号を送信します。

【5】常時起動しておきたい場合
Windowsの「タスクスケジューラ」に登録すると、PC起動時に自動実行され、
落ちても自動で再起動できます。
また、電源設定で「スリープ」を「なし」に、「カバーを閉じたときの動作」も
「何もしない」にしておいてください(画面オフ自体は問題ありません)。

【トラブル時】
- 何もターミナルに表示されない → ボーレートが違う可能性(9600/115200を試す)
- 途中でエラーで止まる → USB接続を確認。再接続すれば自動で復帰します
- 環境変数関連のエラーが出る → 一度 set 変数名= で空にしてから、
  1行ずつ確実にEnterを押しながら設定し直してください
